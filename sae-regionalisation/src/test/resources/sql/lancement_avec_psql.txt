// Sous Linux
// ----------------------

# Pré-requis

=> créer un fichier /root/.pgpass
=> y mettre la ligne suivante :
	localhost:5432:regio:postgres:LE_MDP_DE_POSTGRES
=> éditer le fichier pg_hba.conf (/var/lib/pgsql/9.3/data)
=| commenter la ligne :
	local   all             all                                     peer
   et ajouter la ligne :
   	local   all             all                                     md5
=> commenter la ligne :
	host    all             all             127.0.0.1/32            ident
   et ajouter la ligne :
   	host    all             all             127.0.0.1/32            md5
=> redémarrer postgresql
	service postgresql-9.3 restart


# Commande à lancer

psql --dbname=regio --echo-all -v ON_ERROR_STOP=1 --file=/tmp/regio/sql_vague3.sql --username=postgres --no-password > /tmp/regio/sql_resultats_regio.txt 2>&1 &
