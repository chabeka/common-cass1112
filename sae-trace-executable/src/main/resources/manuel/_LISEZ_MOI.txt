Ligne de commande :
java -jar -Dlogback.configurationFile=[logback] -Doperation=[type_purge] sae-trace-executable.jar [configuration] PURGE [type_purge]
ou
java -jar -Dlogback.configurationFile=[logback] -Doperation=[type_journalisation] sae-trace-executable.jar [configuration] JOURNALISATION [type_journalisation]

avec :
[logback] :			le fichier de configuration des traces
[configuration] : 	le fichier de configuration général du SAE
[type_purge] : 		le type de registre impacté par la purge. Les valeurs possibles sont :
					- PURGE_TECHNIQUE
					- PURGE_EXPLOITATION
					- PURGE_SECURITE
					- PURGE_EVT
[type_journalisation] : le type de journal à produire. Les valeurs possibles sont :
               - JOURNALISATION_EVT
