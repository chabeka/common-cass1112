Pour lancer les tests unitaires, il faut avoir paramétré la variable 
d'environnement BIRT_HOME avec le chemin du BIRT Report Engine.

Exemple : BIRT_HOME = C:\BIRT\birt-runtime-2_5_2\ReportEngine\

Le ReportEngine est disponible sur :
\\cer69-donnees\TECHNOCOM\produits\Outils\Eclipse\Birt\BIRT Runtime\
