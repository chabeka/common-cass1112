Plusieurs commandes sont disponibles :
- listeOrgs :	lister les organismes présents dans l'ensemble des documents
- listeDocs : 	lister les documents dont au moins un des codes organisme est 
			  	présent dans le fichier de correspondance.
- maj       :	mise à jour des documents avec les correspondances des codes
				organismes

				
// listeOrgs
commande :
java -jar -Dlogback.configurationFile=[logback] sae-regionalisation-fond-documentaire.jar listeOrgs [connexions] [sortie]
avec :
[logback] 		= fichier de configuration des logs logback
[connexions] 	= fichier de configuration des connexions DFCE + CASSANDRA
[sortie]		= fichier où sera inscrit les données


// listeDocs
commande :
java -jar -Dlogback.configurationFile=[logback] sae-regionalisation-fond-documentaire.jar listeOrgs [connexions] [sortie] [corresp]
avec :
[logback] 		= fichier de configuration des logs logback
[connexions] 	= fichier de configuration des connexions DFCE + CASSANDRA
[sortie]		= fichier où sera inscrit les données
[corresp]		= fichier properties contenant la liste des correspondances de codes organismes
