Lancement de l'exécutable :

java -jar pdfa-validator-0.0.1.jar [cheminVersPDF] [cheminVersLog] [-h]
	
	avec :
	
	[cheminVersPDF] = chemin complet vers le répertoire contenant les fichiers PDF à analyser
	
	[cheminVersLog] = chemin complet vers le fichier de log, vérifier que le fichier log n'existe pas
	
	[-h] 			= afficher l'aide, h,-help, help et ? peuvent aussi être utilisés.
