Projet contenant des tests d'intégration GNT/GNS
================================================

sae.integration.auto
Tests qui peuvent être lancés automatiquement. Ils se chargent de la mise en place du contexte (par exemple : archivage de documents
nécessaires aux tests), puis du nettoyage


sae.integration.manual
Tests à lancer manuellement. Ils peuvent nécessiter un contexte à gérer manuellement
