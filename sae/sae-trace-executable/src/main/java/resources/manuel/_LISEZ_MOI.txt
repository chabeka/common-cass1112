Ligne de commande :
java -jar -Dlogback.configurationFile=[logback] sae-trace-executable.jar [configuration] PURGE [type_purge]
avec :
[logback] :			le fichier de configuration des traces
[configuration] : 	le fichier de configuration général du SAE
[type_purge] : 		le type de registre impacté par la purge. Les valeurs possibles sont :
					- PURGE_TECHNIQUE
					- PURGE_EXPLOITATION
					- PURGE_SECURITE
