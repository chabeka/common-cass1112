Ligne de commande :

Pour mettre à jour le RND
java -jar -Dlogback.configurationFile=[logback] sae-rnd-executable.jar MAJ_RND [configuration]

Pour mettre à jour les correspondances (modification des documents)
java -jar -Dlogback.configurationFile=[logback] sae-rnd-executable.jar MAJ_CORRESPONDANCES_RND [configuration]

avec :
[logback] :			le fichier de configuration des logs
[configuration] : 	le fichier de configuration général du SAE
